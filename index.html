This test consists of four exercises, each scored on a 0-100 scale:

  • Push-Ups: Linear scoring based on max reps in 2 minutes, rounded down to the nearest 0.5.

Final Score = (Pull-Up Score + Push-Up Score + Plank Score + Run Score) / 4.
